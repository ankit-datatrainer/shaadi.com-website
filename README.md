# Shaadi.com Matrimonial Matchmaking Platform (Frontend)

This is the frontend (React + Vite) for the Shaadi.com-inspired Matrimonial Matchmaking MVP.

## Tech Stack
- React (Vite)
- TailwindCSS
- React Router v6
- Axios
- React Hook Form
- Zod validation
- Context API (Auth + Socket)
- Socket.io client

## Project Structure
- components/
- pages/
- layouts/
- context/
- hooks/
- services/
- socket/
- utils/
- App.jsx
- main.jsx

## Setup
1. Install dependencies:
   ```
   npm install
   ```
2. Start development server:
   ```
   npm run dev
   ```
